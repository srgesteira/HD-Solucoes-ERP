# Checklist browser — Combobox, Entrada Y, MDe, Fatia G

1. **Combobox** — reconcile NF-e, composição (produto componente), PCP (criar OP), almox (saída manual): escolher produto por digitação/lupa.
2. **Entrada Y** — PC → botão «Conferido / finalizar recebimento» em Compras → card em «Recebido» no Faturamento → `/faturamento/entrada/[id]` → Aplicar regras → Finalizar conferência → aba «Finalizado».
3. **Tipo de nota** — PV → Fiscal define tipo (NFS-e / NF-e produto / industrialização) → Pronto para emissão → Expedição «Emitir nota» **sem** perguntar tipo.
4. **MDe** (se conta Focus tiver produto) — admin «Sincronizar SEFAZ» → coluna NF recebidas → Conciliar abre reconcile pré-carregado. Se sync devolver 402/mensagem MDe, activação depende do contrato Focus.
5. **Fatia G** — emissão em **homologação** Focus; produção só com OK explícito do dono.
